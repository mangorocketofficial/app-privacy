## GitHub Pages 활성화
1. GitHub에서 이 repo를 생성
2. 본 폴더(`_privacy_pages/`)의 모든 파일을 repo 루트로 push
3. Settings → Pages → Source: `main` branch / `/ (root)` → Save
4. 잠시 후 위 URL에 접속하여 동작 확인
